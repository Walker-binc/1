Mac Demo Guide

Sales Digital Business Cards for Mac

This package is prepared for Mac-based customer demos and includes both the phone QR flow and the full desktop demo flow.

Important First Step

  1. Do not scan QR images directly after unzip.
  2. You must start the local demo first so the QR code can point to this Mac's current local address.

Recommended Mobile Demo

  1. Unzip the package.
  2. Double-click Mac-手机扫码演示.command.
  3. Wait for the QR page to open in the browser.
  4. Make sure the phone is connected to the same Wi-Fi as this Mac.
  5. Scan the QR code shown in the browser page.

Full Desktop Demo

  1. Double-click Mac-完整后台演示.command.
  2. The browser will open the main team page.
  3. Open the Admin page to edit profile basics, product categories, and page modules.

If the Phone Still Cannot Open the Page

  1. Turn off VPN on the Mac and test again.
  2. Make sure the phone is scanning the new QR code from the browser page, not an older saved image.
  3. Try entering the shown local URL manually on the phone.

If macOS Blocks the Command File

  1. Right-click the .command file and choose Open.
  2. If needed, use Terminal and run chmod +x on the command file.
